# Вебзастосунок для вивчення іноземної лексики

Цей проєкт — вебзастосунок для збереження, повторення та розширення словникового запасу під час вивчення іноземної мови. Застосунок дозволяє користувачу створювати власний словник, повторювати слова, переглядати статистику навчання та використовувати мовну модель штучного інтелекту для отримання контексних прикладів речень і рекомендацій нової лексики.
Програма розроблена як дипломна робота по спеціальності 122 «Комп'ютерні науки» у Львівському національному університеті імені Івана Франка.

## Основні можливості

- додавання, редагування та видалення слів;
- пошук по словнику і сортування словника;
- повторення слів у двох напрямках: англійська → українська та українська → англійська;
- перегляд статистики повторення слів по дням та напрямкам;
- генерація прикладів речень для слів під час повторення за допомогою штучного інтелекту;
- аналіз словника та рекомендації нових слів на його основі — також за допомогою мовної моделі.

### Алгоритм повторення слів

Для вибору наступного слова під час повторення використовується алгоритм, який аналізує кількість правильних і неправильних відповідей для кожного слова, а також коригує коефіцієнт для недавно доданих слів. На основі цих даних формується пріоритет показу слів: менш засвоєні слова пропонуються частіше, тоді як добре засвоєні повторюються рідше, але періодично повертається для закріплення.

### Інтеграція штучного інтелекту

Реалізована за допомогою OpenAI API. Використовує модель GPT з відповідними інструкціями для генерації текстових відповідей.

- **Генерація прикладів речень**: Під час повторення слова застосунок може автоматично згенерувати приклад його використання у реченні. Речення створюється відповідно до напрямку повторення.

- **Аналіз словника та рекомендації нових слів**: Застосунок дозволяє отримати коротку характеристику словника та пояснення щодо можливих напрямків покращення. Аналіз включає визначення тематик доданих слів та рівень знань. На основі цього пропонується список рекомендованих нових слів із перекладом та можливість одразу додати слово до словника.

## Використані технології

- **Frontend**: React, React Router, Tailwind CSS, shadcn/ui, Lucide React, Recharts
- **Backend**: PHP
- **База даних**: PostgreSQL
- **Інтеграція ШІ**: OpenAI API

## Структура проєкту

Проєкт поділено на декілька основних частин.

### `frontend/`

Містить:
- сторінки словника, повторення, статистики та додавання слів;
- маршрутизацію застосунку;
- компоненти інтерфейсу;
- інтеграцію з OpenAI API;
- логіку роботи з AI-функціоналом.

Основні модулі AI:

- `exampleSentence.ts` — генерація прикладів речень;
- `suggestNewWords.ts` — аналіз словника та рекомендації нових слів;
- `responsesCommon.ts` — спільна логіка взаємодії з OpenAI API.

### `backend/`

Відповідає за:
- обробку HTTP-запитів;
- роботу зі словником користувача;
- оновлення статистики повторення;
- взаємодію з базою даних;
- авторизацію та допоміжну бізнес-логіку.

### База даних

Використовується для зберігання:
- користувачів;
- слів та перекладів;
- статистики повторення;
- результатів навчальної активності.

Також використовуються SQL-механізми для підтримки цілісності даних та автоматичного оновлення пов’язаних записів.
